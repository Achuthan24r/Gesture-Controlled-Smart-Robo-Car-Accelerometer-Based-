## Description
This project allows controlling a robot car using hand gestures.
An accelerometer detects hand movement and sends data wirelessly
to the robot using NRF24L01.

## Technologies Used
- Arduino
- MPU6050 Accelerometer
- NRF24L01 Wireless Module
- L298N Motor Driver

## Working Principle
1. Hand tilt data is read using MPU6050
2. Data transmitted wirelessly
3. Robot moves according to gesture direction

## Applications
- Assistive robotics
- Defense robots
- Human-machine interaction
- Hackathons & exhibitions

## Future Enhancements
- AI gesture learning
- OLED feedback
- Voice + gesture combo
